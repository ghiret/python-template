# Documentation Conventions
- Use Google-style docstrings (not NumPy or Sphinx style)
- MkDocs Material theme is configured; use its admonition syntax for callouts
- API docs are auto-generated by mkdocstrings; ensure docstrings are complete
- Keep docs/index.md as the landing page
- Use relative links between doc pages
